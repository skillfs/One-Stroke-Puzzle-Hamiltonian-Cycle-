Algorithm

1.สร้างกราฟกริด N×N + diagonal แบบสุ่ม และตรวจ connectivity ด้วย BFS
2.หา Hamiltonian Path ด้วย Backtracking + Warnsdorff heuristic (เลือก neighbor ที่มีทางออกน้อยที่สุดก่อน)
3.รับประกันว่าทุกด่านมีเฉลย — ถ้าหาไม่ได้ภายใน 5 ครั้ง จะสร้างกราฟใหม่
4.ผู้เล่นต้องลากผ่านทุก node โดยไม่ซ้ำ และต้องมี edge เชื่อมระหว่าง node คู่นั้น
